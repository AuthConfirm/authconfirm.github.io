# AuthConfirm Docs

Welcome to the documentation for **AuthConfirm**, the standard fallback authentication method for [Scratch JP](https://scratchjp.org/)!

## Getting Started

To get started with AuthConfirm, you need to understand how the manual authentication flow works. Because Scratch cloud variables can sometimes be unusable, AuthConfirm offers a manual, comment-based confirmation as a fallback.

### Authentication Flow

Here is the standard flow for a user authenticating:
1. The user asks one of the moderators to authenticate them through ModMail.
2. The moderator tells a code to the user.
3. The user goes to the AuthConfirm Scratch project and pastes the code into the project's comment section.
4. The moderator checks the comment section to verify the user.
